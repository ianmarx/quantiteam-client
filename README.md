# QuantiTeam
QuantiTeam is a web platform for athletes and coaches to track performance data in a team environment.

## Installation
```
$ yarn install
```

## Testing
```
$ yarn test
```

## Run the app in localhost
```
$ yarn start
```

## Deploy to **staging.quantiteam.com**
```
$ yarn deploy-staging
```

## Deploy to **quantiteam.com**
```
$ yarn deploy-prod
```
